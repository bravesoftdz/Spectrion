Autotune procedure

Purpose

  The purpose of this procedure is to provide the user with auto-tuning functionalities.

Optimization page

Optimization method radio button

  The user can choose between the following methods:

  1. Initial seek (in blind). Conditions: There is no signal visible, so the autotune will try all combinations of the set control device parameter/channel values.
  2. Consecutive scanning. Conditions: The signal is visible, but the noise is more than 5% of the signal.
  3. Simplex optimization. Conditions: The signal is visible and the noise is less than 5% of the signal. Simplex is much faster than consecutive scanning, but require stable signal (good S/N ratio)

Control device combo box

   Selects the control device which will be tuned for signal optimization.

Control device’s Channel number control

   Sets the control device’s channel number. For the correspondence channel -> device parameter, see the help for each particular device.

From control field

   Set the starting value for the device parameter/channel to be tuned.

To control field

   Sets the finishing value for the device parameter/channel to be tuned.

Step control field

   Sets the step value between value in From to To.

Number of control devices radio button

   Sets the number of control devices to be tuned. Depending of the autotune method the device's parameters are used in combination (first and third methods) or consecutively (second method).

Measurement device combo box

   Selects the measurement device where the signal will be taken from.

Measurement device’s Channel number control

  Set’s the measurement device’s channel number.

Before each measurement the synchro device is advised for synchronization, so the user can cancel the procedure from there. The setting of synchro device should be about Delay 500 to assure the new parameter value is established before the next measurement is taken.

Intensity radio button

  The user can choose between total and peak intensity for signal to be optimized upon. If peak is chosen the peak in the peak limits is fit with gauss function and the amplitude is used for intensity optimization and peak width - for resolution optimization.

Peak controls

  Set the interval within the peak will be searched for. These are required when the resolution is optimized or peak intensity is selected.

 Intensity/Resolution control

  Sets the intensity/resolution weight. Mathematically only one value can be optimized at the time, so if the user wish to optimize the sensitivity and resolution at the same time he must specify the ratio of importance between these two.

Verification page

At verification page the quality of the signal is verified using as reference previous measured best quality possible mass-spec.

From the button on the top the user can choose reference mass-spec.  From the group bellow the measurement device is set. After the execution of the procedure the "distance" between measured and reference mass-spec is calculated by the same formula used in NIST'98 library search.

 

On the bottom of the autotune window there is a utility for saving / reloading the setting of the procedure.
