Run menu (procedures)

Autotune

    Before you start your measurements you probably would like to estimate the quality of your signal and optimize it along the way.

    This procedure will optimize/find your signal using three different methods depending on the initial conditions (do you have a signal and if you do how good?).

Simple scan

    If you would like to measure your signal (serial mode) as a function of one parameter (or time) here is the place to go.

Uni-scan

    In case you need more complicated measurements as measuring the one or two signals as function of 0, 1 or 2 parameters (control device channels) use uni-scan. The versatility of the procedure will fit most of your scanning needs.

PCA classification

    If you need to classify your samples by their mass spectra against a training set (created in PCA detective - included here). The classification type is one-class classifier (the unknown is tested against each class consecutively) and the method is Principal Component Analysis / Mahalanobis Distance.

The procedures are complemented by two utilities which are simplified scans and are attached to two button on the main tool bar. See options to how set measuring devices for them.

These two utilities are:

    Oscilo - simulating oscilloscope showing in high speed spectra from measurement device in package mode.

    Acquire - similar to Oscilo but keeping the signal in memory, show it as GC/MS and offer a possibility to save it later.
