Uni-scan

I tried to make universal scan as universal as it could be one scan. 

The user has a choice of making change one or two control parameters of control devices and take data from one or two / single or multi channel measurement devices. Always choose the number of control and measurement devices first. All the scanning parameters can be saved in file for later use. All the scanning parameters can be saved in a file '*.scn' extension and retrieved from the disk using the list and two button on the bottom of the uni-scan frame.

Typical tasks are optimizing ion-optics voltages or following long-term stability of the instrument.

The developer can call this procedure using a scan file name as parameter into Execute call and so execute a scan saved beforehand.
