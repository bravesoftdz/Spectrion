Options

Save options radio buttons

  The user can choose between not saving (NO), saving now (NOW) or saving on program exit (On Exit).

On Start tab

Switch devices to On/Off modes.

  On start Spectrion will try to establish connection the the devices checked here. All unchecked devices will be leave Off-line until the user try to connect them manually from right-top button of each device.

Auto Run Script

  The script selected runs automatically on program startup.

   The script appearing in the edit box runs automatically on program startup.

Spectra Path edit box

  Default store data acquisition files in the directory shown in the edit box.

 

Interpreter tab

  Text in Interpreter panel radio buttons

  The user can choose between the following choices:

·        Only the manual commands

·        And the remote commands

·        And the replies from the measurement devices

·        And the replies from control devices

·        And all other replies

    Allow commands/replies visibility to be controlled by remote command check box

 

Templates tab

    Template for New plot, click on Browse button

    Template for Open plot, click on Browse button

    Template for Data Acquisition, click on Browse button

 

    Measuring Devices for Oscilo and Acquire utilities (the default channel is used)

User device tab

    From this tab you can configure additional to the default (Msr; Ctrl and All Dvcs) tabs of the device rack.

    Customized (user) tabs show some and hide other devices. The hiding can be complete or just minimizing (option - Only Minimize Dvcs).

    You can Add or Delete tabs from there and configure the tab when it's active.
